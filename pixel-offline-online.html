📖 User Manual
Getting Started
Prerequisites
Node.js 18+ — Required for the bridge server
Lively Wallpaper — Runs the pet as your desktop wallpaper
Anthropic API key — Optional, enables online AI chat (get at console.anthropic.com)
One-Click Setup (Windows)
1. Double-click SETUP.bat — it does everything automatically:
   ✓ Checks Node.js is installed
   ✓ Installs bridge dependencies
   ✓ Adds bridge to Windows startup
   ✓ Starts the bridge server
2. Open Lively Wallpaper → click + → select pixel-offline-online.html
Done! Your pet lives on your desktop and the bridge auto-starts on every login.
Manual Setup
1. cd bridge && npm install
2. node bridge/server.js or double-click start-bridge.bat
3. Add pixel-offline-online.html in Lively Wallpaper
Connection Status
DotMeaning
🟢 GreenBridge running + Claude AI online
🟡 AmberBridge running + offline AI mode
🔴 RedBridge not running (works offline)
The wallpaper works 100% without the bridge — you still get all widgets (except system monitors), offline AI chat, pet behaviors, themes, and layout customization.

Pet Behaviors
All behaviors can be toggled on/off in Settings → 🐾 Pet tab.
Sleep
Pet falls asleep after your mouse is idle (default: 30 seconds). Shows "Zzz 💤" bubble and breathing animation. Wakes on any mouse movement. Adjustable delay in settings.
Click Reactions
1 click: Random happy message
3 rapid clicks: Gets angry, runs to opposite side
5 rapid clicks: Excited zigzag run
3 screen clicks (1.5s): Runs to your click spot with zigzag path
Hide (Peek-a-boo)
Pet disappears periodically (default: every 5 min for 8 sec), then reappears at a random spot with a peek-a-boo message. Interval and duration adjustable.
Playdate
1–5 random buddy pets visit for ~20 seconds. Three play styles:
Chase: Pets take turns chasing each other
Ball: Volleyball with hit detection
Dance: Circular formation with rotating positions

Mouse interactions during playdates:
• Fast mouse → pets scatter in fear
• Slow approach → pets gather curiously
• Click → pets freeze with surprise
App Watch (Bridge Required)
Monitors your active window and reacts contextually. Detects 20+ app categories:
Browsers, code editors, terminals → Encouragement
Games → Excited "let me play too!"
Music/video players → Watching together
AI tools (ChatGPT, Copilot) → Jealous reactions
Productivity, email, shopping → Contextual comments
15-second cooldown between reactions.
Wander
Pet randomly moves to different spots on screen with smooth easing and a bounce on arrival.
Layout & Positioning
Quick Move (Long-Press)
Hold any widget for 500ms to grab it, then drag anywhere on screen. Release to drop. Works without entering layout edit mode. Does not activate on buttons, inputs, textareas, or links.
Layout Edit Mode
1. Open Settings (Ctrl + ,)
2. Go to 🐾 Pet tab → click ⊞ Edit Layout
3. A 20px grid overlay appears
4. Drag any widget to reposition (snaps to grid)
5. Use 8 resize handles (corners + edges) to resize
6. Click Done to save and exit
Widget Sizing
Scale widgets (most) — Resizing scales content proportionally
Reflow widgets (Chat, To-Do, Sticky, Habits, Quick Links, Countdown, World Clock) — Resizing changes actual width/height, text reflows
Display Settings
If widgets appear off-screen, go to Settings → 🖥️ Display tab:
• Set width/height manually (800–7680 × 400–4320)
• Click ↺ Reset to detected to auto-detect your screen
Reset Layout
Settings → 🐾 Pet tab → ↺ Reset Layout restores all widgets to default positions and clears all saved data (todos, habits, notes, etc.).
All widget positions auto-save and persist across sessions.

Chat & AI Features
Online Mode (Claude AI)
Requires bridge server running + ANTHROPIC_API_KEY set in bridge/.env.
Full Claude AI conversation with personality. Pet responds in character, automatically detects mood from responses, and can comment on your PC hardware stats.
Offline Mode (Built-in AI)
Always available, no requirements. 200+ built-in responses including:
Jokes, facts, motivational quotes, riddles, compliments, time-aware greetings, and contextual responses. Your pet's personality shines through even offline.
Chat Features
Quick pills: Preset message buttons below chat input
File search: Natural language queries to find files on your PC
Unread dot: Red indicator when pet sends message while chat is closed
Scroll: Drag to scroll through chat history
Click the pet to open/close the chat panel.

In Settings → 🐾 Pet tab:
Status dot: Shows connection state

State Persistence
Widget data is saved in two places:
1. Browser localStorage — immediate, works offline
2. Bridge state file (~/.pixel-pet/state.json) — persists across wallpaper reloads
Troubleshooting
Debug Panel
Press Ctrl + D to toggle the debug panel (bottom-left). Shows bridge status, app watch state, pet state, playdate info, registered behaviors, and live active window data.
Common Issues
ProblemSolution
Pet not visibleCheck Lively Wallpaper is running. Click desktop. Check display width/height in Settings
Settings button missingPress Ctrl+, to open directly. Go to Display tab → Auto-Detect
Widgets not showingSettings → 📦 Widgets tab — toggle them on
Bridge not connectingInstall Node.js, run npm install in bridge folder, start with node server.js
Chat says "Offline"Start bridge + set ANTHROPIC_API_KEY in bridge/.env. Offline mode still works without it
Hardware shows no dataBridge must be running. First reading takes 3–5 seconds
Weather shows "--°C"Allow location access, or set city manually in Settings
App watch not reactingBridge required (Windows only). Check debug panel (Ctrl+D)
Pet stuck / not movingClick the pet. Triple-click desktop to call it to you
PowerShell popup flashUpdate to latest bridge/server.js and restart bridge
High RAM usageUpdate to latest bridge (uses persistent process). Restart bridge
Widget won't moveLong-press 500ms on widget background (not on buttons/inputs)
Factory Reset
Settings → 🐾 Pet tab → ↺ Reset Layout clears all positions, widget data, resets pet to Pixel, and theme to Midnight.
